# 🎮 Tic-Tac-Toe Game

A simple and interactive **Tic-Tac-Toe Game** built using **HTML, CSS, and JavaScript**.

This project is a classic two-player game where players take turns placing **X** and **O** on a 3×3 grid. The first player who successfully matches three symbols in a row, column, or diagonal wins the game.

## 🎯 How To Play

1. Player **O** starts the game.
2. Click on any empty box to place your symbol.
3. Players take turns placing **O** and **X**.
4. The first player to get three symbols in a row, column, or diagonal wins the game.
5. Use the **Reset Game** button to restart the game.
6. Use the **New Game** button to start a new match.
